====================================================================
  AUTOMATED TRADING SYSTEM — Getting Started Guide
====================================================================

Read this file first. It walks you through setup and your first run
in under 10 minutes.

--------------------------------------------------------------------
STEP 1: Install Python 3.13+
--------------------------------------------------------------------

Check your version:

    python --version

If you need to upgrade, download from https://www.python.org/downloads/

--------------------------------------------------------------------
STEP 2: Create and activate a virtual environment
--------------------------------------------------------------------

    python -m venv venv

On Mac / Linux:
    source venv/bin/activate

On Windows:
    venv\Scripts\activate

Your prompt will show "(venv)" when the environment is active.

--------------------------------------------------------------------
STEP 3: Install dependencies
--------------------------------------------------------------------

    pip install -r requirements.txt

This installs pandas, numpy, yfinance, matplotlib, and the rest.
Takes 1-3 minutes on a fresh environment.

--------------------------------------------------------------------
STEP 4: Verify the installation
--------------------------------------------------------------------

    python -c "import pandas; import yfinance; print('Ready')"

You should see "Ready". If you get an import error, re-run Step 3.

--------------------------------------------------------------------
STEP 5: Run your first backtest
--------------------------------------------------------------------

    python examples/01_simple_backtest.py

This backtests AAPL over a one-year window and prints a full
performance summary to the terminal.

--------------------------------------------------------------------
STEP 6: Save charts and a Markdown report
--------------------------------------------------------------------

    python examples/05_advanced_analysis.py

Output goes into a timestamped folder:

    runs/AAPL_<YYYYMMDD>_<HHMMSS>/
        report.md                — full performance report
        chart_indicators.png     — price + all indicators
        chart_signals.png        — buy/sell signal overlay
        chart_performance.png    — equity curve and P&L
        chart_risk.png           — drawdown and R:R ratios

Open report.md in any Markdown viewer or text editor.

--------------------------------------------------------------------
STEP 7: Try the other examples
--------------------------------------------------------------------

    examples/02_multi_stock_comparison.py   — compare AAPL/MSFT/GOOGL
    examples/03_custom_configuration.py     — change risk parameters
    examples/04_signal_generation.py        — real-time signal demo

Run them the same way:
    python examples/02_multi_stock_comparison.py

--------------------------------------------------------------------
STEP 8: Explore the code
--------------------------------------------------------------------

Start with these files in order:

    src/automated_trading_system.py   — main entry point
    src/signal_generator.py           — how signals are scored
    src/trading_system.py             — portfolio and risk logic
    src/indicator_calculator.py       — technical indicator math
